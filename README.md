🏠 House Price Prediction using Linear Regression

A machine learning project that predicts house prices using the Housing Dataset and Linear Regression from Scikit-learn. The project includes data preprocessing, categorical feature encoding, model training, evaluation, visualization, and coefficient analysis.

📌 Project Overview

This project demonstrates a complete machine learning workflow:

Load and inspect the dataset
Handle categorical variables using Label Encoding
Split data into training and testing sets
Train a Linear Regression model
Evaluate model performance using regression metrics
Visualize Actual vs Predicted prices
Analyze feature importance using model coefficients
